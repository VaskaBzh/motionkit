# Project Rules

> Короткие, обязательные к исполнению правила проекта. Загружаются автоматически при /aif-implement.

## Rules

- В импортах и экспортах `.ts` файлов использовать расширение `.ts`, а не `.js`
- При импорте из директории не указывать `index.ts` явно — только путь к папке (например `'../types'`, а не `'../types/index.ts'`)
